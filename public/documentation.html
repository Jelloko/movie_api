MyFlix API Documentation

This API provides users with information about various movies. Users can sign up, update their personal information, and manage a list of their favorite movies.

API Endpoints

Business Logic URL Method Description Request Body Response
Welcome to MyFlix! / GET Website greeting None res.send website greeting
Get All Movies /movies GET Returns a list of all movies None JSON object holding data about all movies
Get All Users /users GET Returns a list of all users None JSON object holding data about all users
Get Movie by Title /movies/:title GET Finds a movie by title None JSON object about a single movie containing (_id, Title, Description, Director, Genre, Image, Featured)
Get User by Name /users/:Name GET Finds a user by name None JSON object about a single user containing (_id, Name, Password, Email, Birthday, FavoriteMovies, __v)
Get Genre by Name /movies/Genre/:genreName GET Finds a genre by name None JSON object about a single genre containing (Name, Description)
Get Director by Name /movies/Director/:directorName GET Finds a director by name None JSON object about a single director containing (Name, Bio, Birth, Death)
Register User /users POST Adds new user A JSON object holding data about the user structed like: { "Name": "Pickle", "Password": "Picky3", "Email": "Picket.Bucket@gmail.com", "Birthday": "2000-06-21" } Confirmation message
Update User Information /users/:Name PUT Updates User Information A JSON object holding data about the user structed like: { "Name": "Pickle", "Password": "Picky3", "Email": "Picket.Bucket@gmail.com", "Birthday": "2000-06-21" } Confirmation message
Add Movie to FavoriteMovies /users/:Name/FavoriteMovies/:moviesID PUT Adds a movie to user FavoriteMovies None Confirmation message
Deregister User /users/:name DELETE Deletes user by name None Confirmation message
Remove Movie from FavoriteMovies /users/:Name/FavoriteMovies/:moviesID DELETE Removes a movie from the users favoitemovies None Confirmation message
