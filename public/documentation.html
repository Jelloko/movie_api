MyFlix API Documentation

This API provides users with information about various movies. Users can sign up, update their personal information, and manage a list of their favorite movies.

API Endpoints

Business Logic URL Method Description Request Body Response
Welcome to MyFlix! / GET Website greeting None res.send website greeting
Get All Movies /topMovies GET Returns a list of all movies None JSON object holding data about all movies
Get Movie by Title /topMovies/:title GET Finds a movie by title None JSON object about a single movie containing (title, director, genre)
Register User /users POST Adds new user A JSON object holding data about the user structed like: { name: Jesse, list: { userName: Jello } } A JSON object holding data about the user structed like: { id:1, name: Jesse, list: { userName: Jello } }
Update User /users/:name/:list/:Username PUT Changes username None Confirmation message
Deregister User /users/:name DELETE Deletes user by name None Confirmation message
